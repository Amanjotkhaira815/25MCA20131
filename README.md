Online Bookstore Management System

A mini project using Python Tkinter and MySQL

 Overview
The Online Bookstore Management System is a desktop-based project designed to manage books, customers, orders, and payments efficiently.
It uses MySQL as the backend and Python Tkinter as the frontend.

⚙️ Tech Stack
	•	Frontend: Python (Tkinter)
	•	Backend: MySQL
	•	Connector: mysql-connector-python
	•	IDE Used: VS Code & MySQL Workbench

 Features
CRUD operations for books and customers
 Order placement and payment processing
 Real-time data updates and reporting view
 ER and DFD diagrams included
